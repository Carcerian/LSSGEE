ROLLBACK PLAN (If Issues Arise)
================================================================================

If compilation fails after changes:
  1. Revert .lfm files to git repository version
  2. Apply changes incrementally (one form at a time)
  3. Test compile after each change

If visual appearance is wrong:
  1. Check color values (may vary by monitor/display profile)
  2. Test on multiple machines if possible
  3. Adjust colors slightly and recompile

If performance degrades:
  1. Check image file sizes (scroll image)
  2. Verify font files aren't corrupting form rendering
  3. Simplify any complex styling if necessary
