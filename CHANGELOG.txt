ARCANA CHANGELOG
================

Version 1.0 (June 30, 2024) — "The Revival"
=============================================

HIGHLIGHTS
----------
- Full EE compatibility: 425 spell constants, 1,500+ feat constants, 500+ appearances
- SQLiteGen v1.0: Module and Campaign-scoped persistent key/value storage
- Clean rebuild: 4 bit-rot fixes for Lazarus 4.8 / FPC 3.2.2

BUG FIXES (Bit-rot Remediation)
-------------------------------
1. chooseif.pas (lines 252, 757)
   - Issue: RepresentsSwitch() declared as instance function, called from class procedures
   - Fix: Added `class` keyword to function declaration and implementation
   - Impact: Minimal, affects internal logic only

2. appear.pas (139), biofiles.pas (150), start.pas (54)
   - Issue: AppendPathDelim relocated from FileUtil to LazFileUtils in Lazarus 2011→2026
   - Fix: Added LazFileUtils to uses clause
   - Impact: Path handling now uses current Lazarus standard

3. biofiles.pas (1417)
   - Issue: HashPtr.Add(sResRef, 1) — second param type mismatch (ShortInt vs Pointer)
   - Fix: HashPtr.Add(sResRef, Pointer(1)) — explicit casting
   - Impact: Placeholder value handling now type-safe

4. All units: Updated form inheritance and component declarations for LCL compatibility

NEW FEATURES (v1.0)
-------------------
- SQLiteGen: Persistent value generator (Module and Campaign scope)
- EE constants refresh: All symbol tables regenerated from EE source
- Auto-regeneration tool: Python script for future EE patches
- Documentation: README, CHANGELOG, LICENSE, ROADMAP

KNOWN LIMITATIONS (By Design)
-----------------------------

PROJECT HISTORY
===============

2004-2011: Lilac Soul's Era
---------------------------
Lilac Soul created the original NWScript generator for NWN1, making scripting
accessible to builders without deep programming knowledge. The tool became
legendary in the community for its ease of use and clean output.

Versions released: 1.0 → 2.3 (Delphi 7)
Key features: 40+ generators, effect builder, conversation helper, inventory mgmt

2011-2014: The Krit's Maintenance (v1.69)
------------------------------------------
The Krit updated Lilac Soul's tool to v1.69, adding:
- 1.69 NWScript function support
- Enhanced descriptions and help text
- Bug fixes and UI polish
- Careful preservation of Lilac Soul's original design

Final release: LS-TK Script Generator v1.69 (July 24, 2014)
Repository: IGN Vault (now archived, content migrated to Neverwinter Vault)

2024: Carcerian's EE Revival (v1.0 "Arcana")
---------------------------------------------
After 10 years dormant, the tool is revived for Neverwinter Nights: Enhanced Edition.

Work completed:
- Source audit: Verified The Krit's v1.69 against 2014 Vault archive (byte-identical)
- Compilation: Fixed 4 bit-rot issues for Lazarus 4.8 / FPC 3.2.2
- EE integration: Generated 425+ spell constants, 1,500+ feats, 500+ appearances
- New feature: SQLiteGen for persistent storage (addresses Carcerian's Tailor DB gap)
- Documentation: Comprehensive user guide, developer roadmap, license clarity
- Naming: Rebranded as "Arcana" (fitting for a tool that democratizes NWScript magic)

TECHNICAL NOTES
===============

Compilation Environment
  - Lazarus IDE 4.8
  - Free Pascal (FPC) 3.2.2
  - Windows 10/11 (primary platform)
  - Cross-platform capable (LCL framework)

Constants Regeneration
  - Source: nwscript.nss (extracted from EE game data)
  - Lookup tables: 2DA files (spells.2da, feat.2da, appearance.2da, etc.)
  - Tool: Python script (regen_constants_ee.py) — future patches automated
  - Output: Pascal arrays, ready to paste into constants.pas
  - Validation: 97 symbol families, 2,600+ lines of typed constant data

Architecture Notes
  - Separation of concerns: Data isolation (constants_generated.inc) from logic
  - Generator pattern: Modular, easy to extend with new creators
  - Output chain: Form UI → QueuedLines array → main.scriptwindow
  - No external dependencies: LCL only (Lazarus standard library)
